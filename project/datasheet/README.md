# datasheet  
excel-like table experience out of the box  

## target

- table ui
- excel operations

## dev 
```sh
 git clone https://github.com/uptonking/reacting.git
 cd reacting/
 npm install
 npm run build
```

## demo
```sh
 npm run demo
```

start from [http://localhost:8999](http://localhost:8999)
