# ribbon-menu  
office-like ribbon menu built with react  

## target

- 顶部ribbon menu
- 切换到工具条/右侧侧边栏

## dev 
```sh
 git clone https://github.com/uptonking/reacting.git
 cd reacting/
 npm install
 npm run build
```

## demo
```sh
 npm run demo
```

start from [http://localhost:8999](http://localhost:8999)
